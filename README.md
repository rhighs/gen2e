# Gen2E

A LLM based playwright code generator, provide natural language instructions and get compiled tests back.

## Quick installs

### gen2e
```bash
npm install @rhighs/gen2e -D
```

### gen2e-cli
```bash
npm install -g @rhighs/gen2e-cli
```

### gen2e-interpreter
```bash
npm install @rhighs/gen2e-interpreter -D
```
